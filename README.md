###   To-Do List Application

A simple command-line To-Do List application built with Python. This project allows users to add tasks, view tasks, and manage their daily activities using Python lists.

##  Features

Add new tasks
View all tasks
Menu-driven interface
Simple and beginner-friendly code
Uses Python Lists and Loops

##  Project Structure

todo-list/
│ ├── todo.py └── README.md

##  Technologies Used
Python 3

##  How to Run

Clone the repository:

Navigate to the project folder:
cd todo-list
Run the program:
python todo.py

Lists

##  Learning Outcome

This project helps beginners understand how to:
Store multiple items in a list
Build menu-driven programs
Work with loops and conditions
Create simple real-world Python applications
